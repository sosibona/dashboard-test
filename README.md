# Installing

Run `npm install`

## Run

1. `npm start`
2. Open `http://localhost:3000` in a web browser.

## The tech stack is:
+ HTML5
+ CSS3
+ Flexbox
+ Sass (Scss)
+ BEM methodology
+ JS ES6
+ ReactJS
+ Redux
+ Material UI
+ Formik
